CSS

CSS是一种对Web文档添加样式的简单机制，是一种表现HTML或XML等文件外观样式的计算机语言，是一种网页排版和布局设计的技术。

一、CSS样式规则

CSS样式由选择器和生成块两部分构成，如果对一个选择符指定多个属性时，用分号将属性分开。

二、CSS选择器

（1）通用选择器

使用通用选择器初始化属性值，便于控制

（2）标签选择器

用来控制标签的应用样式。例如：标签选择器p可以用来控制所有p标签的风格

（3）类选择器

使用类选择器，来达到某些特定元素的样式设置

  • 类的名称可以是任意英文字符串，也可以是英文字母和数字的组合
  • 类选择器定义的样式可以被不同的元素同时使用

（4）id选择器

id选择器要求只能在网页中使用一次

可以使用HTML标签的id属性来引用类选择器

（5）伪类选择器

用于向某些选择器添加特殊的效果，最长应用在标签”a”上，表示超链接的4种不同状态。

（6）组合选择器

后代选择器 空格隔开
子元素选择器 “>”号隔开
相邻兄弟选择器 +
普通相邻兄弟选择器 ~

三、网页中应用CSS的4种方式

（1）内联式

直接在HTML标签中使用style属性

（2）嵌入式

将CSS代码添加到HTML文档的head标签之间，并且用style标签进行声明

（3）链入外部样式

通过link标签将外部存放的CSS样式表链接到该页面中，该语句必须存放到head之间

（4）导入外部样式

两种外部样式的区别

  • 隶属关系不同
  • 加载时间以及顺序不同
  • 兼容性不同
  • DOM模型控制样式

样式优先级从高到低分别是：内联->内嵌->外链->外联

五、CSS页面布局

分为固定式，流式子，响应式

（1）border边框

  • 可以按照上->右->下->左顺序一起设置
  • 3个参数时，按照上->左加右->下的顺序
  • 2个参数时，按照上+下->左加右的顺序
  • 一个参数时，四个边框宽度相同

（2）元素的定位

  • CSS右三种基本的定位机制：普通流、浮动和绝对定位
  • 相对定位是将元素相对于它在标准文档流中的位置按照指定便宜移动到新位置，移动后元素形状不变，原本位置所占的空间也继续保留
  • 绝对定位是将元素依据最近的已经定位的父元素进行定位，若所有的父元素都没有定位，则依据body根元素进行定位，移动后元素形状不变，原位置所占空间不保留
  • 固定定位是绝对定位的一种特殊形式，以浏览器窗口作为参照物定义网页元素

（3）弹性盒布局

display的属性有很多：none(不显示)、block(块级元素)、inline(内联元素)、inline-block(行内块元素)、list-item(列表显示)、table(块级表格)、inline-table(内联表格)、flex(弹性布局显示)

2、外链式：

1、数据类型

  • 数值数据number：JavaScript中没有整数和浮点数之分
  • 文本数据string：由单引号或者双引号括起来
  • 布尔数boolean
  • 数组array
  • 函数function
  • 对象objects
  • 无定义数据类型undefined

2、变量命名：

  1. 必须字母或下划线开头，中间是数字字母下划线
  2. 不能包含空格、加、减
  3. 不能使用关键字和保留字
  4. 区分大小写
  5. 变量声明：var

由于JavaScript采用弱类型，因此可以不理会变量数据类型，可以把任意类型的数据赋值给变量

3、关系运算符

  • ==支持自动类型转换，只要两个变量值相同就返回true，忽略类型比较
  • ===必须值相同并且数据类型一致时返回true

4、函数

形参和实参：

  • 在未调用函数时，形参不占用存储单元，只在调用时为形参分配空间，调用结束后自动释放。
  • 形参必须是声明的变量，实参可以是常量、变量或表达式。
  • 在函数调用中，实参列表中参数的数量、类型、和顺序与形参列表中的参数可以不匹配。若形参个数大于实参，多出来的形参值为undifined;反之，多出来的实参将被忽略。
  • 实参对形参的数据传递是单向传递，即实参>形参

系统内部函数：

isFinite 检查某个值是否为有穷大的数
isNaN 检查某个值是否为数字
pareInt 解析一个字符串，返回一个整数
parseFloat 解析一个字符串，返回一个浮点数

四、JavaScript对象编程

1、BOM模型

浏览器对象模型是用于描述浏览器中对象与对象之间层次关系的模型，提供了独立于页面内容，并能够与浏览器窗口进行交互的对象结构

2、DOM模型

文档对象模型( Document Object Model, DOM) 属于BOM的一部分，用于对BOM中的核心对象document进行操作，它是一种与平台、语言无关的接口，允许程序和脚本动态地访问或更HTML、XML文档的内容、结构和样式，且提供了一-系列的函数和对象来实现访问、添加、修改及删除操作。

DOM将网页中文档的对象关系规划为节点层级，构成它们之间的等级关系，这种各对象间的层次结构被称为节点树。

3、事件机制

  • JavaScript是基于对象(Object Based)的语言，而基于对象的基本特征就是采用事件驱动(EventDriven)，在图形界面环境下使得输入变的简单。

  • 事件(Event) 是指可以被JavaScr ipt侦测到的行为(ACTION)。

  • 事件驱动(Event Driver)是指由事件引发的一连串程序的动作。

  • 事件处理程序(Event Handler)是指对事件进行处理的程序或函数。

  • 静态绑定：在HTML标签中，使用事件句柄为该元素绑定指定的事件处理函数

  • 动态绑定：通过JavaScript脚本获得文档中的某一对象，然后通过事件句柄为该元素绑定指定的事件处理函数
